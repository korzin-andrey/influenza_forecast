# Start of the work


To start the work, please, create and a separate virtual environment by running the following strings 
of code 

------------------------------------------------------------------
Linux:

1) ```python -m venv .env ```
2) ```source .env/bin/activate ```
3) ```python -m pip install -e . ```
4) ```export PYTHONPATH=.```
5) ```python gui/app.py ```
------------------------------------------------------------------
Enjoy forecasting influenza outbreaks!
